#### 远程仓库操作

##### 1. 创建远程仓库别名

###### 基本语法

`git remote add 别名 远程地址`

`git remote -v` 查看当前所有远程仓库地址别名

![](C:\Users\南极星\AppData\Roaming\marktext\images\2023-03-15-00-31-57-image.png)

##### 2. 推送本地库到远程库

###### 基本语法

`git push 别名 分支`  需要登录到远程库

##### 3. 拉取远程库到本地库

###### 基本语法

`git pull 别名 分支`

##### 4. 克隆远程库到本地

###### 基本语法

`git clone 远程地址` 克隆不需要登录
